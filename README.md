## 🧠 What is Pages for Mac?

Pages is Apple’s native document editor designed to provide a smooth and simple writing experience. It combines document creation with beautiful templates and seamless Apple ecosystem integration.

It is designed to:

* Design resumes and letters
* Build ebooks and PDFs
* Use professional templates
* Collaborate with others in real time
* Sync across Apple devices via iCloud
